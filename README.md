# Warehouse Automation System (Python)

This project is a simple Warehouse Automation System built using Python and Object-Oriented Programming (OOP). The system helps manage warehouse inventory by tracking goods movement, generating inventory reports, and forecasting demand for products.

## Features

* Add goods to warehouse inventory
* Remove goods from inventory
* Track goods movement
* Generate inventory reports
* Forecast product demand

## Technologies Used

* Python
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository or download the project.
2. Open the project folder.
3. Run the Python file.

```
python warehouse_system.py
```

4. Choose options from the menu to manage warehouse inventory.

## Learning Purpose

This project demonstrates how Python classes and functions can be used to build a simple automation system for warehouse inventory management.
